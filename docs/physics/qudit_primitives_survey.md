**Pass**: pure research. No Sturm source code touched.
**Status**: ground-truth PDFs in `docs/physics/`; implementation-side decisions deferred to Tobias + a follow-up design pass.

This note fixes notation, collects the canonical theorems for universality on $(\mathbb{C}^d)^{\otimes n}$, and evaluates three candidate continuous 1-parameter families to generalise Sturm's $q.\theta$ and $q.\phi$ primitives. Every non-trivial claim cites a specific equation or theorem in a locally-downloaded PDF.

All conventions below use $\omega = e^{2\pi i/d}$ unless noted.

## 1. Universality target

### 1.1 Weyl–Heisenberg generators (the target the root-of-unity knob must recover)

From **Gottesman 1998, quant-ph/9802007** (`gottesman_1998_qudit_fault_tolerant.pdf`), §2, p.3:

$$X_d|j\rangle = |j+1 \bmod d\rangle, \qquad Z_d|j\rangle = \omega^j|j\rangle, \qquad X_d Z_d = \omega^{-1} Z_d X_d. \tag{G4}$$

(Eq. (4) in Gottesman.) The operators $X_d$ and $Z_d$ are the canonical qudit "shift" and "clock" operators; they generalise the qubit $X,Z$ Paulis. They have order $d$ (p.3 after Eq. G5).

The same operators are called "displacement operators" in the stabiliser/MSD literature and written $D(x|z) = \tau^{xz} X^x Z^z$, $\tau = \omega^{(p+1)/2}$ (**Howard–Vala 2012, 1206.1598**, Eq. (3), p.2, `howard_vala_2012_qudit_magic.pdf`).

### 1.2 Universality theorem (our north star)

**Brylinski–Brylinski 2002, quant-ph/0108062**, Theorem 1.3 (p.2, `brylinski_brylinski_2002_universal_quantum_gates.pdf`):

> Let $V$ be a 2-qudit gate. Then the following are equivalent:
> (i) $\{\text{all 1-qudit gates}\} \cup \{V\}$ is universal (dense in $U(d^n)$);
> (ii) $\{\text{all 1-qudit gates}\} \cup \{V\}$ is exactly universal (generates $U(d^n)$);
> (iii) $V$ is imprimitive (does not map every product state to a product state).

**Consequences** (p.3 after Eq. (2.1)):

- The *generalised CNOT* $X|ij\rangle = |i, i\oplus j\rangle$ (addition mod $d$) is imprimitive, since it turns $(|0\rangle+|1\rangle)\otimes|0\rangle$ into $|00\rangle+|11\rangle$. Hence $\{U(d)^{\otimes n},\ X_{\text{SUM}}\}$ is exactly universal for every $d\ge 2$.
- The generalised CNOT is exactly the SUM gate from Gottesman Eq. (G12): $|i\rangle|j\rangle \to |i\rangle|(i+j)\bmod d\rangle$.

**Brennen–Bullock–O'Leary 2005, quant-ph/0509161** (`brennen_bullock_oleary_2005_efficient_qudit_circuits.pdf`), §3, p.3 after Eq. (6):

> We prove for the first time that $U(d)^{\otimes n} \sqcup \{\text{CINC}\}$ is exact-universal.

where $\text{CINC}|c,t\rangle = |c, (c+t)\bmod d\rangle$ when $c=d-1$ (their convention: control fires on $|d-1\rangle$). This is equivalent up to basis relabelling to the Brylinski SUM gate. Compared to Muthukrishnan–Stroud's two-parameter $\Gamma_2[X_d],\Gamma_2[Z_d]$ library, Brennen–Bullock–O'Leary's result is strictly stronger: **one** two-qudit gate suffices when we allow arbitrary local $U(d)$.

### 1.3 Clifford + one non-Clifford for fault tolerance (Gottesman's closure)

**Gottesman 1998**, §6 Eq. (45), p.8: the qudit Toffoli

$$|a\rangle|b\rangle|c\rangle \to |a\rangle|b\rangle|(c+ab)\bmod d\rangle \tag{G45}$$

together with the Clifford group (generated by SUM + Fourier $R$ + phase $P$ + scalar $S$) gives universal fault-tolerant computation on stabiliser codes for prime $d$ (Gottesman's main Theorem, stated informally in the abstract and §6). For $d=2$ this reduces to the Shor/Gottesman {Clifford, Toffoli} set.

**Practical takeaway for Sturm**. Three separate things are "universality":
1. Unitary-on-$(\mathbb{C}^d)^{\otimes n}$ universality (Brylinski): needs any imprimitive two-qudit gate + any faithful continuous 1-qudit family.
2. Clifford group (Gottesman, §2 pp.3–4): generated by SUM + Fourier + phase.
3. Fault-tolerant universality (Gottesman §6): Clifford + qudit-Toffoli.

The primitive pairs evaluated below must support (1); (2) falls out automatically; (3) is the fault-tolerance question and is orthogonal to continuous-vs-discrete single-qudit parametrisation.

## 2. Candidate A — continuous Weyl–Heisenberg ($X^\alpha,\ Z^\beta$)

The first-instinct extension: exponentiate the Gottesman generators.

### 2.1 Matrix form

Let $\hat{Q} = \sum_{j=0}^{d-1} j\,|j\rangle\langle j| = \mathrm{diag}(0,1,\dots,d-1)$ be the "position" operator, and $\hat{P} = F^\dagger \hat{Q} F$ the "momentum" operator, where $F$ is the $d$-dimensional QFT (Fourier gate $R$ of Gottesman Eq. (G10)). Then

$$Z^\beta := e^{-i\beta \hat{Q}} = \sum_j e^{-i\beta j}|j\rangle\langle j|, \qquad X^\alpha := e^{-i\alpha \hat{P}} = F^\dagger e^{-i\alpha \hat{Q}} F.$$

$Z^\beta$ is diagonal in the computational basis; $X^\alpha$ is diagonal in the Fourier basis.

Explicitly in the computational basis, with $\omega = e^{2\pi i/d}$,

$$\langle j'|X^\alpha|j\rangle = \frac{1}{d}\sum_{k=0}^{d-1} \omega^{-k(j'-j)} e^{-i\alpha k}.$$

At $\alpha = 2\pi/d$ this collapses to a Kronecker delta: $\langle j'|X^{2\pi/d}|j\rangle = \delta_{j',(j+1)\bmod d}$ — the shift operator $X_d$ of Eq. (G4). Done directly, since the eigenvalues of $\hat{P}$ are exactly the integers $0,1,\dots,d-1$ (same as $\hat{Q}$, just in a different basis), so $e^{-i(2\pi/d)\hat{P}}|k\rangle_P = \omega^{-k}|k\rangle_P$, where $|k\rangle_P = F^\dagger|k\rangle$.

### 2.2 Reduction to qubit Ry/Rz at $d=2$

At $d=2$, $\hat{Q} = \mathrm{diag}(0,1) = \tfrac{1}{2}(I - \sigma_z)$. Then

$$Z^\beta = e^{-i\beta \hat{Q}} = e^{-i\beta/2}\, e^{i\beta \sigma_z/2}.$$

Up to the global phase $e^{-i\beta/2}$, $Z^\beta = R_z(-\beta)$. Sturm's $q.\phi\ \text{+=}\ \delta$ is $R_z(\delta)$, so the correspondence is $\delta = -\beta$ (or equivalently define $Z^\beta = e^{+i\beta\hat{Q}}$ to remove the sign).

Global phase is irrelevant by project policy: **CLAUDE.md "Global Phase and Universality"** section — Sturm lives in SU($d$), and $U$ vs. $e^{i\gamma}U$ define the same channel $\rho\to U\rho U^\dagger$. Nielsen–Chuang §4.2 Eq. (4.4) makes the same point: a global phase has no observable effect. (Reference only — not quoting equation from the DJVU.)

For $X^\alpha$, by the QFT conjugation $X^\alpha = F^\dagger Z^\alpha F$ and $F = H$ at $d=2$, so $X^\alpha|_{d=2} = H R_z(-\alpha) H = R_x(-\alpha)$ up to global phase (Nielsen–Chuang §4.2 Eq. (4.8) identity $HR_z H = R_x$).

**This is $R_x$, not $R_y$.** That matters: Sturm's amplitude primitive is $R_y$. The fix: pick the $y$-conjugate. Define

$$\hat{P}_y := F_y^\dagger \hat{Q} F_y, \quad F_y := \exp(-i\pi \hat{J}_y/2)\ \text{(spin-}(d-1)/2\text{)},$$

i.e. use a "$Y$-Fourier" instead of the standard Fourier. At $d=2$, $F_y = R_y(-\pi/2)$, and conjugation of $\hat{Q}$ by $F_y$ yields $\sigma_y$. Then $X^\alpha_Y := e^{-i\alpha \hat{P}_y}|_{d=2} = R_y(-\alpha)$ up to global phase.

**Bottom line**: to recover Sturm's $R_y$ (not $R_x$) at $d=2$, candidate A needs a $y$-twisted QFT, not the standard one. This is a harmless but real design choice — and it is not idiomatic in the qudit literature, which universally uses the $x$-shifting $X_d$. If Sturm exposes $q.\theta$ as "$R_y$-flavoured shift" and $q.\phi$ as "clock phase", at $d=2$ we recover exactly the current qubit primitives; at $d>2$ the $y$-shift has no Weyl–Heisenberg name but exists.

### 2.3 Root-of-unity recovery

By construction (§2.1): $X^{2\pi/d} = X_d$ (shift), $Z^{2\pi/d} = Z_d^{-1}$ (clock, sign convention). Equivalently $X^{2\pi k/d} = X_d^k$ and $Z^{2\pi k/d} = Z_d^{-k}$, covering all WH Pauli powers.

### 2.4 Universality with primitive #4

Choose #4 = SUM $|i,j\rangle \to |i, (i+j)\bmod d\rangle$ (Gottesman Eq. G12 / Brylinski §2 p.3). SUM is imprimitive (Brylinski §2, $X$-example p.3: $(|0\rangle+|1\rangle)|0\rangle \to |00\rangle+|11\rangle$). By Brylinski Theorem 1.3 (iii⇒ii), $\{U(d),\ \text{SUM}\}$ is exactly universal on $(\mathbb{C}^d)^{\otimes n}$.

But we don't get *all* of $U(d)$ from $\{X^\alpha, Z^\beta\}$ — this is a 2-parameter abelian-ish family. To get dense $U(d)$ we need at least one more non-commuting 1-qudit gate. The standard trick (Brennen–Bullock–O'Leary, §5.1 "Householder") is to combine $X^\alpha$ with its $F$-conjugate (i.e. $Z^\beta$) and iterate. **Claim we need to verify**: does the free group generated by $\{X^\alpha, Z^\beta \mid \alpha,\beta\in\mathbb{R}\}$ densely fill $SU(d)$?

For $d=2$, yes: $\{R_x(\alpha), R_z(\beta)\} = \{X^\alpha, Z^\beta\}$ is Euler-angle-complete (Nielsen–Chuang §4.2). For $d\ge 3$, the generators are 1-dimensional subgroups of $SU(d)$ isomorphic to $U(1)$, meeting at the identity; Lie bracket $[i\hat{Q}, i\hat{P}]$ gives a third direction, and iterated brackets should span the Cartan + one root, but not all of $su(d)$ — $su(d)$ has dimension $d^2-1 \ge 8$ for $d\ge 3$ while $\hat{Q},\hat{P}$ span a 2d Cartan. **This is a gap.** Candidate A alone does not reach dense $U(d)$; we need either (a) another 1-qudit generator, or (b) a more generous #4 (e.g. controlled-$X^\alpha$ across a continuous parameter — see open question Q1).

The fix is not fatal: add, e.g., $F$ (Fourier) as a closed-form discrete 1-qudit gate, or add a second continuous family in a non-Cartan direction (e.g. Howard–Vala's $T$-analogue, or Brennen–Bullock–O'Leary's $\wedge_1(V)$ controlled-Householder which is already 1-qudit-local after the control is fixed).

### 2.5 CV limit (P7)

As $d \to \infty$ with $j\in\{0,\dots,d-1\}$ relabelled $j = x\sqrt{d}$ for continuous $x$, $\hat{Q} \to \hat{x}$ and $\hat{P} \to \hat{p}$ (Weyl's construction, canonical treatment in Nielsen–Chuang Ch. 7 / quantum optics textbooks). Then $X^\alpha \to e^{-i\alpha \hat{p}}$ and $Z^\beta \to e^{-i\beta \hat{x}}$ — the **displacement operators** on $L^2(\mathbb{R})$, which are exactly the Heisenberg–Weyl generators for CV.

**Bartlett–deGuise–Sanders 2002, quant-ph/0109066** (`bartlett_deGuise_sanders_2002_qudit_simulation.pdf`), p.2 between Eqs. (10)–(11), makes a caveat: in the number-basis encoding via Fock states, the naive $d\to\infty$ limit gives the Pegg–Barnett phase operator, not the Heisenberg–Weyl $\hat{x}$. Direct quote (p.2):

> The number operator $\hat{N}$ and the phase operator $\hat{\theta}_z$ are conjugate in the same sense that momentum and position are conjugate, but the limit does not yield the usual continuous–variable Pauli group: the Heisenberg–Weyl group, with position $\hat{x}$ and momentum $\hat{p}$ operators as generators.

So candidate A's "clean" CV limit depends on the computational basis chosen. If the basis is the Fock number basis, you get Pegg–Barnett phase ($\theta$-operator), which has known pathologies at $d\to\infty$ (non-self-adjoint position-conjugate operator). If the basis is relabelled-position (i.e. qudits as discretisations of the line, as in GKP codes), you get the honest Heisenberg–Weyl generators. **The Sturm P7 commitment to Gaussian CV means choosing the latter interpretation** — and this constrains the physical meaning of $|0\rangle,\dots,|d-1\rangle$ on a Sturm qudit at large $d$. Flag for future design: whether $|j\rangle$ is "quanta count" or "lattice coordinate" is a distinction without a difference for finite $d$ circuits but becomes load-bearing at the CV boundary.

### 2.6 P9 / Bennett compatibility

Sturm's Bennett.jl compiles classical typed functions $f: \mathbb{Z}_{2^W} \to \mathbb{Z}_{2^W}$ to reversible circuits by CNOT + Toffoli. For qudits, the analogous arithmetic target is $\mathbb{Z}_d^W$ (width-$W$ registers of base-$d$ "trits"/"dits"). The reversible universal set in $\mathbb{Z}_d^W$ is SUM + qudit-Toffoli (Gottesman Eq. G45). The continuous $X^\alpha,Z^\beta$ primitives don't directly enter the Bennett target — they're the *analogue-gate* part of the DSL, used for controlled unitaries/phase kickback. So Bennett's compilation target changes (Toffoli-qudit instead of qubit-Toffoli) regardless of which continuous pair we pick. Candidate A is compatible; the compilation-target change is forced by P7, not by the choice of A/B/C.

## 3. Candidate B — spin-$j$ $su(2)$ ($\hat{J}_x,\hat{J}_y,\hat{J}_z$)

A qudit of dimension $d = 2j+1$ is the spin-$j$ irrep of $SU(2)$. The generators $\hat{J}_x,\hat{J}_y,\hat{J}_z$ are $d\times d$ Hermitian matrices satisfying $[\hat{J}_a,\hat{J}_b] = i\epsilon_{abc}\hat{J}_c$.

### 3.1 Matrix form

Standard Wigner representation: in the $|j,m\rangle_z$ basis ($m=-j,\dots,j$),

$$(\hat{J}_z)_{mm'} = m\,\delta_{mm'}, \quad (\hat{J}_\pm)_{mm'} = \sqrt{j(j+1)-m'(m'\pm1)}\,\delta_{m,m'\pm1}, \quad \hat{J}_\pm = \hat{J}_x \pm i\hat{J}_y.$$

Continuous rotations: $R_a(\delta) := e^{-i\delta \hat{J}_a}$, $a\in\{x,y,z\}$.

Sturm's qubit primitives at $j=1/2$ ($d=2$): $\hat{J}_z = \tfrac{1}{2}\sigma_z$, $\hat{J}_y = \tfrac{1}{2}\sigma_y$. Then $R_y(\delta) = e^{-i\delta\hat{J}_y} = \cos(\delta/2)I - i\sin(\delta/2)\sigma_y$ — exactly Sturm's $q.\theta \mathrel{+}= \delta$. Similarly $R_z(\delta) = q.\phi \mathrel{+}= \delta$. **No sign-or-phase convention acrobatics required.** This is Candidate B's killer feature.

### 3.2 Root-of-unity recovery

The eigenvalues of $\hat{J}_z$ are $-j, -j+1, \dots, j$, not $0,1,\dots,d-1$. So $e^{-i(2\pi/d)\hat{J}_z}$ has spectrum $\{\omega^{-m}\}_{m=-j}^j = \{\omega^j,\omega^{j-1},\dots,\omega^{-j}\}$. For $d$ odd ($j$ integer), this is exactly $\{1,\omega,\dots,\omega^{d-1}\}$ modulo a global $\omega^{-j}$. For $d$ even ($j$ half-integer), there is a $\sqrt{\omega}$ offset. Either way, with a $Z_d^{-1}\omega^j$ relabelling, $R_z(2\pi k/d)$ is the clock operator $Z_d^k$ up to global phase.

For $R_x$, the spectrum of $\hat{J}_x$ is the same (Wigner matrices are unitarily related). The offset gets absorbed into the "phase-state" basis Bartlett Eq. (12) (p.3): in the $|j,m\rangle_x$ basis, the discrete Fourier transformed states,

$$|s\rangle := \tfrac{1}{\sqrt{d}}\sum_{m=-j}^j e^{2\pi i m s/d}|j,m\rangle_x, \quad (d \text{ odd})$$

Bartlett Eq. (13):

$$X_d|_{\text{odd }d} = \exp(2\pi i \hat{J}_x/d), \quad X_d|_{\text{even }d} = \exp(-i\pi/d)\exp(2\pi i \hat{J}_x/d).$$

So root-of-unity $\hat{J}_x$ rotation reproduces $X_d$, with a parity-dependent global phase.

### 3.3 Universality with primitive #4

**Known obstruction**: $su(2) \subsetneq su(d)$ for $d > 2$. The subgroup $SU(2) \hookrightarrow SU(d)$ (via spin-$j$ irrep) has dimension 3, whereas $SU(d)$ has dimension $d^2-1$. At $d=3$: 3 < 8. At $d=4$: 3 < 15. You cannot reach dense $U(d)$ with only $\{R_x(\alpha),R_y(\beta),R_z(\gamma)\}$.

**Bartlett's resolution** (`bartlett_deGuise_sanders_2002_qudit_simulation.pdf`, p.3 Eq. (17) and surrounding text):

> To perform arbitrary unitary transformations on a single oscillator efficiently, one may employ a combination of linear optics, squeezing, and a nonlinear process such as photon detection [7] or a nonlinear optical Kerr interaction [14]. (p.3 col 2)

Translated: single-qudit $SU(d)$-completeness needs a **non-linear Hamiltonian on the spin** — i.e. a term polynomial of degree $\ge 2$ in $\hat{J}$ (e.g. $\hat{J}_z^2$). Equivalently, in the spin-$j$ picture, adding one element of the squeezing algebra $\mathfrak{sp}(2)$ (generated by $\hat{J}_z^2$ and friends) breaks $su(2)$ up to $su(d)$. The SUM gate alone is still imprimitive and closes universality on $(\mathbb{C}^d)^{\otimes n}$ **once** we have all of $SU(d)$ on one qudit. Bartlett Eq. (17) gives an explicit 4-wave-mixing Hamiltonian $\chi \hat{N}_1 \hat{N}_2$ implementing SUM.

So Candidate B needs: $\{R_x(\alpha), R_z(\beta), \text{one nonlinear gate like } e^{-i\gamma \hat{J}_z^2}, \text{SUM}\}$. This is a 4-primitive set, not 3.

### 3.4 CV limit

Beautiful and canonical. $d\to\infty$ with $j = (d-1)/2 \to \infty$ and Holstein–Primakoff transformation

$$\hat{J}_+ \to \sqrt{2j}\,\hat{a}, \quad \hat{J}_z \to j - \hat{a}^\dagger\hat{a}$$

takes $\hat{J}_x,\hat{J}_y \to \sqrt{2j}\,(\hat{x},\hat{p})$ (up to scaling), reproducing $L^2(\mathbb{R})$ canonical commutation. Bartlett p.2 col 2 Eqs. (9)–(10) set this up: with computational basis $|n=s\rangle_{HO}$ and $\hat{N}|n\rangle = n|n\rangle$, the qudit becomes a truncated Fock space; the $d\to\infty$ limit is rigorous on the number operator but NOT on $\hat{J}_x$ directly (the Pegg–Barnett phase caveat noted in §2.5). For smooth CV, choose the Holstein–Primakoff chart and restrict to low-excitation subspace.

### 3.5 P9 / Bennett compatibility

Same as Candidate A: Bennett.jl's target shifts to qudit-Toffoli. The 1-qudit continuous primitives are orthogonal to Bennett.

### 3.6 Summary of B

Strongest mathematical match to Sturm's existing Ry/Rz at $d=2$ (no conventions to fiddle with). Loses single-qudit universality for $d>2$ without a nonlinear partner. Cleanest CV limit.

## 4. Candidate C — Gell-Mann / $su(d)$ Cartan + one off-diagonal

The straightforward choice in the qudit literature, both Muthukrishnan–Stroud and Brennen–Bullock–O'Leary.

### 4.1 Matrix form

$su(d)$ has a Cartan subalgebra of dimension $d-1$ (diagonal traceless Hermitian matrices) and $d(d-1)$ off-diagonal root generators $E_{jk} = |j\rangle\langle k| + |k\rangle\langle j|$, $F_{jk} = -i(|j\rangle\langle k| - |k\rangle\langle j|)$ for $0 \le j < k \le d-1$, and Cartan $H_{k} = |k\rangle\langle k| - |k+1\rangle\langle k+1|$. The Gell-Mann basis for $d=3$ gives the 8 traceless Hermitian $\lambda_1,\dots,\lambda_8$.

**Brennen–O'Leary–Bullock 2004/5, quant-ph/0407223** (`brennen_2005_criteria_exact_qudit_universality.pdf`), abstract p.1:

> We consider a generic physical system in which the single qudit Hamiltonians are a small collection of $H^x_{jk} = \hbar\Omega(|k\rangle\langle j| + |j\rangle\langle k|)$ and $H^y_{jk} = \hbar\Omega(i|k\rangle\langle j| - i|j\rangle\langle k|)$.

i.e. **pairwise** $x$ and $y$ rotations on 2-level subspaces $\{|j\rangle,|k\rangle\}$. These generate the "ladder" of $R_x,R_y$ gates on every 2-level slice of the qudit. Together with one Cartan rotation ($R_z$-style) and a 2-qudit entangler, they generate $SU(d^n)$.

Wang–Hu–Sanders–Kais 2020 (`wang_hu_sanders_kais_2020_qudit_review.pdf`), Eq. (4) p.4:

$$X_d^{(l)}(x,y) = \mathrm{diag}(I_{l-1}, U_2(x,y), I_{d-l-1}), \qquad U_2(x,y) = \frac{1}{\sqrt{|x|^2+|y|^2}}\begin{pmatrix}x & -y\\ y^* & x^*\end{pmatrix}.$$

This is a 2-parameter family acting as a $U(2)$-rotation on the $\{|l-1\rangle,|l\rangle\}$ subspace. Together with $Z_d(\theta)$ (Wang Eq. (5) p.4: $e^{i\theta}$ on $|d-1\rangle$, identity elsewhere) and the controlled extension $C_2[R_d]$ (Wang Eq. (6) p.5), the set $\Gamma_d = \{X_d^{(l)}, Z_d, C_2[R_d]\}$ is universal (Wang Eq. (11) p.5).

### 4.2 Reduction to qubit Ry/Rz at $d=2$

At $d=2$, there is only one 2-level subspace and $l$ can only be $1$:

$$X_2^{(1)}(x,y) = U_2(x,y) = \frac{1}{\sqrt{|x|^2+|y|^2}}\begin{pmatrix}x & -y\\ y^* & x^*\end{pmatrix}.$$

This is a generic element of $U(2)$ parametrised by 2 complex numbers (4 real, minus 1 constraint after normalisation = 3, matching $U(2)$'s dimension when counting phases; for $SU(2)$ we take $|x|^2+|y|^2=1$). Setting $x = \cos(\delta/2), y = \sin(\delta/2)$ gives $R_y(\delta)$. Setting $x=\cos(\delta/2), y=-i\sin(\delta/2)$ gives $R_x(\delta)$.

So candidate C's 1-qudit primitive at $d=2$ is generically $U(2)$, specialising to $R_y$ or $R_x$ by parameter choice. This matches Sturm's $q.\theta\mathrel{+}=\delta$ when we set $(x,y) = (\cos(\delta/2),\sin(\delta/2))$ on the appropriate 2-level block.

### 4.3 Root-of-unity recovery

Candidate C is not naturally a "root-of-unity recovers $X_d$" construction — $X_d^{(l)}$ only acts on one 2-level block. To build the full cyclic shift $X_d$ of Gottesman Eq. (G4), chain $d-1$ transpositions: $X_d = P(0,1)\cdot P(1,2)\cdots P(d-2,d-1)$ where $P(j,k)$ is a single Pauli-$X$ on the $\{j,k\}$ block. This is Muthukrishnan–Stroud Eq. (20) (p.3, `muthukrishnan_stroud_2000_multivalued_logic.pdf`) and Brennen Eq. (7) (p.3, `brennen_bullock_oleary_2005_efficient_qudit_circuits.pdf`).

So root-of-unity values don't directly yield $X_d$; one needs a **discrete product** of $d-1$ primitives. **This is a real downside vs. Candidate A**, which hits $X_d$ at a single rotation angle.

### 4.4 Universality with primitive #4

Brennen–Bullock–O'Leary §3, p.3 after Eq. (6):

> We prove for the first time that $U(d)^{\otimes n} \sqcup \{\text{CINC}, \text{CINC}^{-1}\}$ is exact-universal.

where CINC is the cyclic increment. Since CINC is imprimitive (same reasoning as SUM), this is Brylinski's theorem.

Brennen–O'Leary–Bullock 2004 (`brennen_2005_criteria_exact_qudit_universality.pdf`) abstract: **even the discrete set of 2-level Hamiltonians $H^x_{jk},H^y_{jk}$ plus one two-qudit Hamiltonian** gives exact universality. This is the cleanest formal statement matching Sturm's idiom of "rotations on 2-level blocks".

### 4.5 CV limit

Ugly. The Gell-Mann basis of $su(d)$ has no canonical $d\to\infty$ limit that lands in the Heisenberg–Weyl algebra. You can embed $su(d) \hookrightarrow su(\infty)$ but the distinguished generators are basis-dependent, and which 2-level blocks "survive" depends on how you arrange them. The Heisenberg–Weyl limit is most naturally the continuous limit of Candidate A ($\hat{Q},\hat{P}$ are each in the Cartan subalgebra of $su(d)$ — but so is every other diagonal matrix).

**Verdict**: Candidate C has no natural CV limit. If P7 is load-bearing (and it is: the PRD says so), C is weakest here.

### 4.6 P9 / Bennett compatibility

Same story: qudit-Toffoli drives Bennett's target, not the choice of continuous primitive.

| **Root-of-unity → WH shift/clock** | Exact: $X^{2\pi k/d} = X_d^k$ in a single rotation | Exact up to parity phase (Bartlett Eq. 13) | Needs product of $d-1$ transpositions |
| **Universality with SUM (#4)** | Needs extra 1-qudit primitive (Fourier, or third continuous family) | Needs one nonlinear 1-qudit gate ($J_z^2$) | Complete (Brennen, Brylinski Thm 1.3) |
| **CV limit ($d\to\infty$)** | Clean if basis = position-eigenstates; Pegg–Barnett pathology if basis = Fock | Cleanest — Holstein–Primakoff gives $\hat{x},\hat{p}$ directly from $\hat{J}_x,\hat{J}_y$ | No natural limit |
| **P9 / Bennett** | Compatible. Bennett target shifts to qudit-Toffoli independently. | Same | Same |
| **"Continuous 1-parameter" count** | 2 (α, β) | 3 (θ_x, θ_y, θ_z), pairs to 2 at d=2 (R_y, R_z) | $d-1$ block angles + $d-1$ phase angles (redundant parametrisation at d=2) |
| **Fits current Sturm idiom?** | Good if we choose $y$-twist; otherwise $R_x$-like | Perfect match for $R_y, R_z$ | Parametrisation overspecified at d=2; matches on a chosen 2-level block |

## 6. Recommendation

**Lean: hybrid A+B.** Expose two continuous 1-parameter primitives $q.\theta \mathrel{+}= \delta$ and $q.\phi \mathrel{+}= \delta$ defined as $R_y$-like and $R_z$-like rotations on the *full qudit* using the spin-$j$ representation (Candidate B mechanism), and expose a *third* discrete family — the qudit Fourier $F$ — as a named gate (not a primitive!) built from Candidate A for root-of-unity access to $X_d$.

**Reasoning (the tradeoff, explicitly)**:

1. **Clean d=2 reduction is non-negotiable** per P7+P8: the existing qubit tests and every derived gate in `src/gates.jl` must keep working unchanged. Candidate B matches exactly; A needs a $y$-twisted Fourier (cosmetic but "not what the literature calls $X^\alpha$"); C over-parametrises.

2. **Single-qudit universality is non-negotiable**: any serious computation needs arbitrary $U(d)$ on one qudit. B needs a nonlinear partner (one extra primitive); A needs an extra generator too; C is already complete but at the cost of a confusing parametrisation.

3. **CV limit is non-negotiable** per P7: B's Holstein–Primakoff route to $\hat{x},\hat{p}$ is the textbook story; A's CV limit exists but depends sharply on basis interpretation; C has no clean CV limit at all.

4. **Root-of-unity one-liners** are nice-to-have: they make $X_d,Z_d$ trivially expressible in user code. A wins here, B and C need a loop/product.

The hybrid: use B's generators for continuous knobs (preserves d=2 story, clean CV limit), add **one** nonlinear 1-qudit primitive — call it $q.\theta_{2} \mathrel{+}= \delta$ for $e^{-i\delta \hat{J}_z^2}$ (quadratic phase / "squeezing" on the spin-$j$ irrep) — to close $SU(d)$ universality, and provide the discrete qudit-Fourier $F$ as a library gate for WH operators via root-of-unity. This is **3 continuous primitives + 1 entangler**, or **5 primitives total** counting preparation — up from 4.

**Alternative if we insist on 4 primitives**: go C alone. Pay the parametrisation cost at d=2 (have $q.\theta$ secretly be a 2-parameter family with a canonical reduction), accept the ugly CV limit, and rely on the Brennen construction as the universality ground truth. This is the "most literature-compatible" option but foreclosure risk on P7-CV is real.

**What I am not recommending**: Candidate A standalone. The d=2 story forces a $y$-twisted Fourier that nobody in the literature uses, and the CV limit is basis-interpretation-dependent — risky for a foundational choice.

## 7. Open questions (for Tobias)

**Q1. Which #4 (two-qudit entangler)?**

Options:
- (a) **SUM** $|a,b\rangle \to |a, (a+b)\bmod d\rangle$ (Gottesman Eq. G12, Brylinski §2): simplest, imprimitive, universality closed.
- (b) **Qudit-Toffoli** $|a,b,c\rangle \to |a,b,(c+ab)\bmod d\rangle$ (Gottesman Eq. G45): three-qudit, needed for fault-tolerance closure on stabiliser codes.
- (c) **Controlled-$X^\alpha$** (continuous controlled shift): richer, direct route to Shor-style phase estimation; Brylinski Thm 1.3 still applies.
- (d) **CINC** (Brennen's cyclic increment): equivalent to SUM under relabelling, but their proof uses this form.

The user-level `a ⊻= b` notation strongly suggests (a) SUM or (d) CINC. Fault-tolerance arguments want (b) to be a library gate, not necessarily a primitive.

**Q2. Global phase convention in $SU(d)$.**

At $d=2$ Sturm's project policy is "live in SU(2), pay the $H^2 = -I$ cost" (CLAUDE.md "Global Phase and Universality"). For $d>2$ with the Holstein–Primakoff or spin-$j$ generators, there is a parity-dependent global phase between $X_d$ built from $R_x(2\pi/d)$ vs. the literature's $X_d$ (Bartlett Eq. (13), `bartlett_deGuise_sanders_2002_qudit_simulation.pdf` p.3: $\exp(-i\pi/d)$ prefactor at even $d$). Do we:
- (i) extend the "we live in SU($d$), no fix-ups" rule and add a test "controlled $X_d$ built from spin-$j$ rotation disagrees with controlled $X_d$ from the displacement-operator formula by a controlled phase"?
- (ii) add a convention knob that normalises the phase depending on parity?

(i) is most honest to the channel-not-unitary stance. (ii) is user-friendly. I suspect (i) is what P5+P7 force.

**Q3. Parity of $d$: prime-only or arbitrary?**

Gottesman 1998 explicitly restricts the fault-tolerance theorem to prime $d$ (abstract, §2 p.3, and comment after Eq. G5: "the structure is somewhat more complicated when $d$ is not prime, and I have not verified that these gates are sufficient for the nonprime case"). Brylinski–Brylinski Theorem 1.3 and Brennen–Bullock–O'Leary both state exact-universality for arbitrary integer $d \ge 2$ — no primality needed for unitary universality. So:
- Unitary universality: any $d \ge 2$. OK.
- Stabiliser codes / FT: prime $d$ only (for now). Library gate restrictions are fine; the type `QMod{d}` should not foreclose $d=4,6,\dots$.

Sturm probably wants to allow any $d \ge 2$ at the type level, and document "QECC on prime $d$ only for v0.1" as a constraint in the QECC module.

## 8. Foreclosure risks I am flagging (not decisions)

- **Anyons (out of scope)**: fusion categories naturally give qudit-like systems with *braiding* instead of continuous rotations. Wang–Hu–Sanders–Kais §2.1.2 (p.6) mentions metaplectic anyons giving a universal qupit set via Cui–Wang — this is a **discrete** gate set, and it fits C's framework but not A's or B's continuous-family framework. If we ever want topological/anyonic Sturm, C has the structural edge. B is not foreclosed but needs additional machinery (continuous braids don't exist in the same way).
- **Infinite-d / Fock / bosonic codes**: CV limit discussion in §2.5, §3.4 above. B is best; A is second; C is not a starter. If Sturm P7 means "Gaussian CV works by the same primitives", rule out C.
- **Prime-composite crossover for QECC**: Gottesman's restriction to prime $d$ is fundamental (divisors of $d$ break the cyclic structure). Sturm's type system should carry primality as a trait, not a type parameter. Deferrable.
